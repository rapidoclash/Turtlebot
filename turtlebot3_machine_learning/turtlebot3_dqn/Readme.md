# Modèles entraînés

Ces modèles marchent pour un lidar projetant 15 lasers de -1.25 à 1.25 radian.

## Pour le réseau neuronal :
- **Couche d'entrée** : 19 neurones
- **Couche suivante** : 25 neurones
- **Couche de sortie** : 5 neurones
